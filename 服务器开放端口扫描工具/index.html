Server-Port-Scan-Tool

服务端基于Node.js开发，可扫描公网/内网服务器开放端口，外网处理速度2分钟，内网处理速度20秒。

功能特点

  • 非阻塞I/O操作，高并发处理
  • 支持公网/内网端口扫描
  • 可选择多种扫描模式
  • 自动解析域名
  • 搭建过程简单

服务器搭建教程

  1. 将端口扫描工具下载至本地，解压缩
  2. 下载 node.js 环境
  3. 安装服务器依赖，执行npm install express
  4. 终端切换至工具所在文件夹，执行node server.js
  5. 浏览器访问 http://localhost:8080
